使用html+css创建静态网页

使用html+css创建静态网页

使用html+css创建静态网页

使用环境

ideaiJ IDEA 2019.2.2

界面功能预览

本静态网页初步模拟了一个招聘网站的功能与界面。

网页分块

网页主要分为headed(头部导航栏)，project（主体），footer（页脚）三个部分。

headed

头部导航栏的设计包括一个logo标志、菜单选项、搜索框和一个登录/注册入口

基础设计：设计导航栏的高度和宽度以及颜色。使用position：fixed使得浏览页面过程中，导航栏始终悬停在顶部

logo图标：插入对应的图片，图片可以是png、jpg、jpeg的格式

在css文件中设置它的位置

菜单栏：

我这里是直接使用了几个超链接，#的位置可以换成要跳转的页面。

同样，在css文件中设置它的格式，不做赘述。

问题1：生成的div标签换行显示在logo图片的下方

解决办法：在之前logo的标签中加上style=”float:left”，即使logo图片右对齐，此时的菜单栏就和logo图片显示在同一行了

问题2：如何控制各个超链接之间的距离

解决办法：在css文件中设置a标签的样式，使用margin-left：20px 控制a标签之间的距离；我这里还设计了字体的样式。

登录/注册：仍然是使用超链接的形式，并用float:left让它居左显示，且使得后续的搜索框也能正常显示

搜索栏：使用input输入框，placeholder显示提示字，再用button设定一个搜索按钮。

product

product分为三个部分，分别是职业大类选择栏，职业具体子菜单，热门企业栏。

background：通过对background的相关的设定使得网页不单调，我选用了一张图片作为网页的背景，大小设定的是全覆盖，并且采用了与导航栏相同的技术fixed使其固定。

职业选择栏：我在这里使用了表格table创建一个纵向菜单，在其中的每一个格子中使用a标签加入超链接。每个链接都导向对应的子菜单，例：

在这里设置了鼠标悬停即所在的格子背景变蓝、字体变白的效果，使用的是hover事件：

职业具体子菜单：使用iframe标签将一个网页（html）嵌入现有的网页（html）中

scrolling=”No”表示不显示网页滚动条

热门企业栏：给这个div标签一个题目，然后插入各企业的图标，写一个点击触发事件，点击不同的企业图标会跳转到对应企业的页面

footer

这个部分只有一个模块，采用经典的盒子模型，然后标注信息，链接，以及版本号。再使用css进行简单的排版即可。
